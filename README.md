# Tree Buddy

Git worktree manager for macOS. Visualize, create, and manage worktrees from your menubar or terminal.

## Requirements

- macOS 15+
- Node.js 20+

## Installation

```bash
npm install
```

## Usage

### CLI

```bash
npm run cli:bun
```

Navigate with `j`/`k`, quit with `q`, open command palette with `/`, help with `?`.

### GUI (Menubar)

```bash
npm start
```

## Development

```bash
npm test        # Run tests
npm run build   # Compile TypeScript
```
